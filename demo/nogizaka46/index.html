Azure Media Cognitive Demo: Video Frames Face Recognition


Video and Closed Caption



Key technologies and features:

  • Face detection and Identification: Speakers' names are displayed on the video closed caption above as the speakers acutally appear, which is achieved basically by the following steps - (1) registering speakers' faces in Cognitive Services' Face API beforehandin, (2) tuning video frames into thumbnails by using Azure Media Services' Media Encorder Standard, and (3) detecting and identifying speakers' faces by using Cognitive Servcies Face API
